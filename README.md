## Public Prototype (Free)

The public release includes:
- 20 isolates
- Binary AMR gene presence/absence matrix
- Clean metadata (country, host, isolation source where available)
- One row per isolate
- One column per AMR gene feature

This subset is intentionally limited and suitable for rapid experimentation.

---

## Why this dataset matters

- African AMR genomic data is severely underrepresented
- Most ML benchmarks rely on non-African datasets
- Region-specific signals are often lost in global aggregates

This dataset enables exploratory and comparative ML analysis using African-derived data.
